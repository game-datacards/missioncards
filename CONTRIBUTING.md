# Contributing

Thank you for considering a contribution. New decks, corrections, layout improvements, and translations are all welcome.

## How to contribute

1. Open an issue describing what you want to add or change, or go straight to a pull request for small fixes.
2. Keep card dimensions and naming consistent with the existing decks (one folder per deck, descriptive kebab-case filenames, separate front and back images).
3. Make sure your contribution is your own work or work you have the right to submit.
