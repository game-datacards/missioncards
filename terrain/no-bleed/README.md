# Terrain Layouts — No Bleed

[Back to all decks](../../README.md)

Tarot sized terrain layouts (70 x 120 mm), no bleed, ready to print and trim at home. Images are 1653 x 2833 px, roughly 600 DPI at card size.

Each terrain layout shows the recommended terrain placement for a mission matchup. Every matchup comes as several alternative layouts, including a portrait-oriented variant. Two sets are provided: one with the movement / placement measurements printed on the layout, and one without.

| Set | Layouts |
| --- | --- |
| [No Measurements](no-measurements/README.md) | 45 |
| [With Measurements](with-measurements/README.md) | 45 |
